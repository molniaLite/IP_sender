Перед запуском программы установить зависимости (pip install requirements.txt) и внести данные в user_data.json
В user_data.json:
    "smtp_server" адрес smtp сервера вашей почты (к примеру smtp.mail.ru);

    "mail_from" адрес электронной почты с которой будет отправляться сообщение;

    "mail_pass" пароль или токен для электронной почты из "mail_from";

    "mail_to" адрес на который будет приходить сообщение;
